# Formato de Fim de Semana - MotoGP

## Sexta-feira
- Treinos livres para ajuste inicial de setup.
- Coleta de dados de pneu, consumo e estabilidade.

## Sabado
- Sessao classificatoria para definicao de grid.
- Corrida sprint em formato reduzido de distancia.

## Domingo
- Warm-up curto para validacao final.
- Corrida principal com pontuacao completa.

## Pontuacao geral
- Corrida principal concentra maior peso de pontos.
- Sprint adiciona pontos estrategicos para campeonato.

## Objetivo competitivo
- Maximizar consistencia em sprint e corrida principal.
- Reduzir risco mecanico sem comprometer desempenho.
